# HM24 Case App

This repo was developed for hm24 case. Built-in Golang.

## Installation

You only need to download go modules.

```bash
go mod tidy
```

## Running Service
This application is a basic api. In main directory, you should execute following command.

```bash
go run cmd/main.go 
```
